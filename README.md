### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/prettyquail/Django-Channels.git
   ```
2. activate virtualenv
3. install requirements.txt file
4. Use WebSocket King and hit the url "ws://localhost:8000/ws/Chat/"
